## API 文档

响应数据示例：

```
{
    "code": xx,
    "data": xx,
    "msg": "xx"
}
```

> 登录接口 url : `/auth` 请求方法：`POST` 请求参数：

|参数名|类型|是否必传|说明|
|:-----:|:-----:|:-----:|:-----:|
|username|string|是|用户名|
|password|string|是|密码|

__备注__: 无

---

> 上传图片接口 url : `/upload` 请求方法：`POST` 请求参数：

|参数名|类型|是否必传|说明|
|:-----:|:-----:|:-----:|:-----:|
|image|string|是|图片名称 |

__备注__: 无

---

> 获取标签列表接口 url : `/api/v1/tags` 请求方法：`GET` 请求参数：

|参数名|类型|是否必传|说明|
|:-----:|:-----:|:-----:|:-----:|
|name|string|是|标签名|
|state|int|否|状态|

请求头：

|名称|类型|是否必传|说明|
|:-----:|:-----:|:-----:|:-----:|
|token|string|是|jwt token|

__备注__: 无

---
